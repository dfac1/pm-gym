## Value Proposition

### Структура
```
┌──────────────────────────────────────────────────────────┐
│              Почему PM Gym?                              │
│                                                          │
│  [ICON]           [ICON]           [ICON]                │
│  Практика         Системная        Измеримый             │
│  без рисков       обратная связь   прогресс              │
│                                                          │
│  Описание         Описание         Описание              │
└──────────────────────────────────────────────────────────┘
```

### 3 ключевые ценности

#### 1. 🎯 Практика без рисков
Учись на ошибках в симуляции, а не на реальном продукте.

#### 2. 💡 Системная обратная связь
Детальный разбор каждого решения с альтернативами.

#### 3. 📊 Измеримый прогресс
Отслеживай развитие конкретных навыков.

## How It Works

### Структура
```
┌──────────────────────────────────────────────────────────┐
│              Как это работает?                           │
│                                                          │
│  1 → 2 → 3                                               │
│                                                          │
│  [STEP 1]      [STEP 2]      [STEP 3]                    │
│  Выбери        Принимай      Получай                     │
│  кейс          решения       фидбек                      │
└──────────────────────────────────────────────────────────┘
```

### 3 шага

#### 1. 📝 Выбери кейс
Выбери продуктовую ситуацию из библиотеки

#### 2. 🤔 Принимай решения
Проходи ветвящийся сценарий, делая выборы

#### 3. 💡 Получай фидбек
Анализируй последствия и учись на разборе
